# Gestion Parc de Voitures

Projet  simulant la gestion d’un parc de voitures.

## Auteur
Nom: Walid
Cours: INF1093 Programmation II

## Description
Ce programme permet de:
- créer des employés
- créer des voitures
- attribuer une voiture à un employé
- retirer une voiture
- empêcher l’attribution d’une voiture déjà utilisée
